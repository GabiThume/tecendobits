Na ordenação com o algoritmo bubble sort, são comparados os pares adjacentes e trocados quando o elemento anterior for maior que o posterior, de forma a ordenar a lista da direita para a esquerda. Embora seja pouco eficiente, é de simples implementação e interessante quando o número de elementos a serem ordenados é pequeno.

Na ordenação com o algoritmo quick sort, é escolhido um elemento para pivô, e a lista é reordenada em elementos menores e maiores do que este pivô. Esse processo é repetido recursivamente para cada uma das partições. É um algoritmo que usa a estratégia de dividir para conquistar.

function merge(left,right,saida){
   var a=0;
   while(left.length&&right.length)
      saida[a++]=right[0]<left[0] ? right.shift(): left.shift();
   while(left.length)
      saida[a++]=left.shift();
   while(right.length)
   saida[a++]=right.shift();
}

function mSort(saida,tmp,l){
   if(l==1) return;
   var m=Math.floor(l/2), tmp_l=tmp.slice(0,m), tmp_r=tmp.slice(m);
   mSort(tmp_l,saida.slice(0,m),m);
   mSort(tmp_r,saida.slice(m),l-m);
   merge(tmp_l,tmp_r,saida);
}

Na ordenação com o algoritmo merge sort, divide-se a sequência original em conjuntos de dois elementos e os ordena, depois os agrupa em conjuntos de quatro elementos e ordena esses conjuntos, aí então agrupa em conjuntos maiores e os ordena até ter uma divisão em apenas duas partes que juntas formam a sequência ordenada. É um algoritmo de dividir para conquistar.

function shellSort (saida) {
   for (var h = saida.length; h = parseInt(h / 2);) {
      for (var i = h; i < a.length; i++) {
         var k = saida[i];
         for (var j = i; j >= h && k < a[j - h]; j -= h)
            saida[j] = saida[j - h];
         a[j] = k;
      }
   }
   return a;
}

shellSort(saida);

Na ordenação com o algoritmo shell sort, os itens separados por um certo número de posições h são rearranjados a partir do primeiro elemento, depois a partir do segundo e assim por diante, quanto todos tiverem passado por essa ordenação inicial o h vai diminuindo até chegar em igual a 1. Separar a sequência em linhas de tamanho igual a h e ordenar pelas colunas facilita o entendimento. Esse algoritmo é uma melhoria do algoritmo de inserção.
