Arduino Temperatura

Published June 24, 2010


Como utilizar o circuito integrado LM35 da National Semiconductors no arduino para medir a temperatura em graus Celsius?

Precisamos de um resistor de 100kohm, de alguns jumpers, do arduino e do LM35, que está custando cerca de R$3,50.

O LM35 possui 3 pinos. Com o chanfro virado para sí, a perna da esquerda é o positivo, a do meio é a tensão e a da direita o negativo.

A voltagem de saída do pino de tensão do sensor é proporcional a temperatura, onde cada 0,01Volt significa 1 grau centígrado.

Ligue a saída volt (tensão) do sensor na porta analógica de número 0 do arduino, para poder ler essa variação. Como a porta analógica converte o que está sendo lido em uma variação de 0 a 1024 e a tensão de referência é de 5volts, a fórmula para a temperatura é (5* valor lido na porta analógica*100)/1024.

A perna + (positivo) vai nos 5v do arduino e o – (negativo) vai pro resistor de 100kohms ao terra e para a porta analógica 0 do arduino.

Código no arduino para ler do sensor LM35 na entrada analógica 0 e transformar isso em graus Celsius:

int pin=0;
int valor;
int temperatura;
void setup(){
  Serial.begin(9600);
}
void loop(){
  valor=analogRead(pin);
  temperatura = (5*valor*100)/1024;
  Serial.println(temperatura);
  delay(100);
}
